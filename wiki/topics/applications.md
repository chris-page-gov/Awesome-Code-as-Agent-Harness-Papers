# Applications

## Scope

The README applies the harness frame to code assistants, GUI/OS agents,
autonomous embodied agents, scientific discovery agents, and an agent
personalization section currently present in this fork.

## Useful Entry Points

- Code assistants: harness evolution, repository memory, benchmarks, CI/CD, and
  shared repositories.
- GUI/OS agents: GUI and OS environments as partially observable program worlds.
- Embodied agents: grounded policies, reusable skills, and auditable deployment.
- Scientific discovery: experiment loops, simulators, self-driving labs, and
  scientific-agent benchmarks.
- Personalization: preference state, memory, and feedback as editable harness
  artifacts.
